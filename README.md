# CampusShare

A full-stack multi-college resource-sharing platform enabling students to borrow, lend, and recover lost items within their campus.

## Features
- OTP-based authentication (college email)
- Multi-college marketplace isolation
- Borrow/Lend workflow with request lifecycle
- Lost & Found module
- Image upload support

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: PostgreSQL
- Auth: JWT + OTP (Nodemailer)
